COUNTRY & TERRAIN PROFILES
═══════════════════════════════════════════════════════════════════════

  Profile 001  Angola_Overland_Profile_2026-04.html        ON GITHUB
               18-section profile (rebuilt Apr 2026 to LC76 v3.0 design system).
               Southern Angola: Lubango, Serra da Leba, Namibe, Baia dos Pipas,
               Baia dos Tigres, Iona NP, Foz do Cunene. Northern circuit: Luanda,
               Kalandula Falls, Pedras Negras. Visa-free SA passport. Yellow Fever
               mandatory. Landmine awareness throughout. Interactive Leaflet maps
               (3 tabs). Portuguese field phrases. Compiled 8 March 2026,
               rebuilt 28 April 2026.

  Profile 002  Namibia_Overland_Profile_2026-04.html        ON GITHUB
               18-section profile. Entry via Vioolsdrif. Kaokoland circuit (Epupa
               Falls, Van Zyl's Pass, Marienfluss, Purros), Caprivi Strip, Zambia
               crossing at Wenela/Sesheke. Vet cordon Red Line, NWR booking
               (email-only Apr 2026), Bwabwata/Mudumu/Nkasa Rupara parks, SADC fee
               tiers. Interactive Leaflet maps (3 tabs). Diesel NAD 23.63/L
               (Apr 2026). Compiled 28 April 2026.

  Profile 003  Botswana_Overland_Profile_2026-04.html       ON GITHUB
               18-section profile. SA passport visa-free 90 days. Entry from
               Namibia Caprivi at Ngoma Bridge. Kalahari focus: CKGR (Deception
               Valley, BigFoot Tours), Makgadikgadi (Khumaga/SKL, Baines'
               Baobabs/Xomae), Kgalagadi. Northern parks: Chobe (Ihaha/Kwalate,
               Savuti/SKL, Linyanti/SKL), Moremi (Third Bridge/Xomae,
               Xakanaxa/Kwalate, Khwai North Gate/SKL). Full 5-operator DWNP
               booking matrix. Park fees: Chobe/Moremi BWP 300/pax/day SADC +
               BWP 138/vehicle; other parks BWP 200/pax SADC + BWP 90/vehicle.
               Diesel BWP 22.66/L (BERA Apr 2026). FMD vet cordon documented.
               Cost estimate 10-night two-person northern circuit: ~BWP 14,671.
               Setswana phrases. Interactive Leaflet maps (3 tabs). Compiled
               28 April 2026.

  Profile 004  Zambia_Overland_Profile_2026-04.html         ON GITHUB
               18-section profile. Entry at Wenela/Sesheke from Namibia Caprivi.
               KAZA UniVisa USD 50. Victoria Falls (Mosi-oa-Tunya), South Luangwa
               (walking safaris, night drives, leopards), Kafue (Busanga Plains),
               Lower Zambezi (canoe safaris). USD cash at all park gates — crisp
               post-2013 notes. DNPW fee structure documented (SADC self-drive:
               SL/LZ USD 20 pppd + USD 15 vehicle). Tsetse fly, bilharzia, unfenced
               camp protocols. Diesel ZMW 29.78/L (ERB April 2026). Compiled
               28 April 2026.

  Profile 005  LC76_Kalahari_Profile_2026-04.html           ON GITHUB
               Terrain/ecosystem profile (not country format). 16 sections.
               Six zones: Kgalagadi Transfrontier (SA+BWA), Mabuasehube (BWA),
               Khutse GR (BWA), CKGR (BWA), Namibian Kalahari corridor (NAM),
               Green Kalahari/Augrabies (SA). Geology: fossil sand sea 900,000 km²
               paradox. San people: 100,000-year lineage, 2006 CKGR land rights
               ruling. Wildlife: black-maned lion (mane mechanism), brown hyena
               (<10,000 globally), gemsbok, cheetah, springbok, black-footed cat,
               meerkat, sociable weaver, puff adder/cape cobra. Trans-Kalahari
               route guide: full south-north corridor ~1,400 km (16–22 days) +
               Variant A CKGR fast circuit + Variant B SA-only Kgalagadi loop.
               Water/fuel matrix: zone-by-zone carry requirements. Booking matrix:
               SANParks (Kgalagadi R134/adult/day SA resident; Augrabies R73/adult/
               day — 2025/26), Intertourism (Mabuasehube BWP 200/pax SADC),
               BigFoot Tours (Khutse + CKGR BWP 200/pax SADC), DWNP entry permits,
               Gondwana (Namibia). Leaflet maps (3 tabs): all zones, trans-Kalahari
               route, CKGR interior. Setswana + Afrikaans phrases. Compiled
               29 April 2026.
