What makes a "holistic" LiFePO4 system
A holistic LiFePO4 system is not just a battery and a charger. It is a fully integrated energy ecosystem consisting of: Battery Bank (storage) → DC-DC Charger (alternator input) → Solar MPPT Controller (solar input) → AC Shore Power Charger (mains input) → Battery Monitor (state awareness) → Distribution & Protection (safe output management) → Outputs (12V, USB-C PD, USB-A, cig socket). Every layer has a job. A weakness in any single layer undermines the whole system.

✓ All components are fully compatible with either the 200Ah or 300Ah battery
The REDARC BCDC1225D, Victron BMV-712, Victron Blue Smart, Freedom Distribution Board, and Egon DC Hub are all sized for charge current and load draw — not battery capacity. Switching between a 200Ah and 300Ah battery requires no changes to any other component. The only setup difference is programming the correct capacity (200 or 300) in the BMV-712 app at installation — a one-time step. Charge times differ because the 200Ah has a smaller deficit to fill, but all charge sources operate identically.
Charge time from 50% SoC at 25A: 200Ah (~3.2 hrs shore / ~3.5 hrs driving) · 300Ah (~4.8 hrs shore / ~5.2 hrs driving). The Ah decision is purely about autonomy and budget — it has no knock-on effect on any other component in either configuration.

What is the Egon DC Hub?
Egon DC Hub 2.1 — centralised power distribution hub, not an all-in-one charger
The Egon DC Hub is a premium centralised distribution board. Think of it as an exceptionally well-engineered fuse box and bus bar combined — every component in your system terminates at one clearly-labelled board with screw-type connectors. The battery, DC-DC charger, shore charger, solar input, fridge, lights, and USB ports all connect at one place. No crimping, soldering, or lugs required.
What it replaces: in a conventional build, an auto-electrician must run individual cables between separate bus bars, fuse blocks, and components — dozens of termination points scattered through the vehicle. The Egon collapses all of that complexity to one board. Each fused position has a built-in LED indicator so any blown fuse is visible at a glance without reaching for a test meter.
What it does NOT include: the Egon has no built-in DC-DC charger, no MPPT solar controller, and no battery monitor. Config B still uses the same REDARC BCDC1225D and Victron BMV-712 as Config A — the Egon provides the hub through which they all distribute power. It is a fully analogue, passive device — no Bluetooth, no app, no software, no electronics that can fail in the bush. That simplicity is a feature, not a limitation.
Why it saves labour: because all cables land at one board, installation time is measurably shorter — the auto-electrician doesn't need to build a custom wiring harness between separate components. This makes Config B modestly more expensive in component cost but partially offsets via lower labour. It also makes future modifications straightforward — one look at the Egon board tells you exactly how the system is wired.

Consultant Recommendation
Both are excellent — choose based on your priority
Choose Configuration A (Victron + Freedom + REDARC) if you are doing extended remote trips in Botswana, Namibia, or Mozambique where you need IP67 protection against dust and water ingress, where you want a physical BMV-712 display in the cabin that requires no phone, and where you value the ability to replace any individual component if one fails 800 km from the nearest town.
Choose Configuration B (Egon DC Hub) if you want a premium distribution board with individual LED fault indicators per output, 10 fused circuits plus two 80A high-current terminals, and the installation clarity of every cable landing at one board. The premium over Config A is approximately R 5,500–R 6,200 in component cost, partially offset by lower installation labour.
For the LC76 SW doing the routes described (KAZA, Namibia, Mozambique): both configurations deliver identical charging performance and monitoring. Config A is lower cost. Config B is the choice if you want a cleaner, more inspectable distribution setup and are happy to pay the premium for it.
