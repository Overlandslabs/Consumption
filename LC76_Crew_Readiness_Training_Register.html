§1 · Register Scope & Method

This is the evidence-based record of demonstrated crew capability for the planned 2028–2030 Southern & East Africa expedition. Every skill carries a status, the evidence that supports it, a sign-off line, and — where it lapses (first-aid, recovery currency) — a renewal trigger. Skills that need crew redundancy are assessed for both crew independently.

The honest picture: the crew is experienced. Mary-Ann has travelled solo to Tanzania and back; both hold recovery, sand and obstacle course training (~2024); terrain skill is a demonstrated strength. So this register does not drill competencies already proven.

The genuine redundancy gaps are narrow and shared by both: the newly-installed Victron LiFePO4 system (limited operating hours for either crew), the comms / escalation protocol (new — the escalation tree is defined in R38 (EX-2)), and lapsed first-aid certification. This register targets those, plus a winch-focused refresh of the ageing recovery training. Status is baked into each row and revised as items complete; the checkbox layer is a working sign-off overlay for planning and drill days.

Status: DEMONSTRATED proven, evidenced, no action  ·  REFRESH DUE currently competent, renewal scheduled  ·  SCHEDULED planned, not yet done  ·  OUTSTANDING gap — must schedule
